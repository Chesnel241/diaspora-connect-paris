# Dossier des photos d'événements

Placez vos photos d'événements ici. Elles apparaîtront automatiquement dans le carrousel sur le site.

## Formats supportés
- PNG (.png)
- JPEG (.jpg, .jpeg)
- WebP (.webp)
- GIF (.gif)

## Recommandations
- **Dimensions optimales**: 1920x1080 pixels (ratio 16:9)
- **Taille maximale**: 500 KB par image pour des performances optimales
- **Nommage**: Utilisez des noms descriptifs (ex: `concert-2024.jpg`, `reunion-paris.png`)

## Exemple
```
events/
├── concert-gospel-2024.jpg
├── reunion-priere-mars.png
├── convention-2023.webp
└── temoignages-avril.jpg
```

Les images défileront automatiquement toutes les 4 secondes sur le site.
